Sites internet, landing pages, maquettes et copies web.

- Copies de landing pages (textes, headlines, CTAs)
- Structures de sites (architecture, arborescence)
- Maquettes textuelles de pages web
- Contenus SEO, meta descriptions, balises
- Briefs pour developpeurs ou designers

Exemples : site MedKit, page de vente IApreneur, landing page Chatflow.
